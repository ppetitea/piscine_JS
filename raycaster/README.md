# piscine_JS # raycaster

A simple javascript raycaster by segment intersection with sectors and portals..

Learnings :
- vertex
- vector
- matrix
- camera
- fisheye correction
- colliding
- segment intersection (thanks to daniel shiffman)
- optimization by using sector and portals
- recursivity
- modules
- js canvas
- css design
